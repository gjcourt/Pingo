# research/

Spikes, investigations, and vendor evaluations.

**Put here:**
- Output of exploratory work — what we learned and why it shaped a later decision.
- Vendor comparisons and proof-of-concept write-ups.

**Do not put here:**
- Decisions — `design/`.
- Stable lookup material — `reference/`.

**Naming convention:** `<yyyy-mm-dd>-<topic>.md`
Examples: `2026-09-01-alternative-dns-providers.md`.

**Allowed `status:` values:** `Complete` (research is frozen once written).

Research docs are not updated after they're written. If the conclusion changes, write a new research doc that supersedes it.
